## Sesión Aeróbica Cognitiva

### Activación Mental (10 minutos)
- Ejercicios calentamiento cognitivo
- Preparación toma decisiones
- Ejercicios seguimiento visual
- Activación enfoque mental

### Entrenamiento Principal - Integración Cognitiva-Aeróbica (35 minutos)

**Entrenamiento Bicicleta Cognitiva:**
- Intervalos Decisión: 6×2 minutos
- Señales visuales durante ciclismo
- Tomar decisiones manteniendo esfuerzo
- 90 segundos recuperación entre intervalos

**Entrenamiento Core Cognitivo:**
- Dead Bugs Patrón: 4×6 cada (seguir patrones visuales)
- Planchas Reacción: 4×30 segundos (responder a señales)
- Descanso: 60 segundos entre series

**Integración Cognitiva Tenis:**
- Sprints Decisión: 6×10 segundos (reaccionar a señales)
- Cambios Dirección Cognitivos: 6×5 cambios
- Descanso: Recuperación completa para claridad mental

### Recuperación Cognitiva (10 minutos)
- Descompresión mental
- Movimientos controlados lentos
- Ejercicios restauración enfoque

## Instrucciones Rápidas Ejercicios

### Intervalos Decisión (6×2 min)
- Mantener esfuerzo ciclismo constante
- Responder a señales visuales o auditivas durante ejercicio
- Tomar decisiones rápidas mientras ejercitando
- Entrena coordinación cerebro-cuerpo
- **Referencia Video**: [Jeff Nippard - Guías Cardio](https://www.youtube.com/watch?v=Kl0ht3ow8TQ)

### Dead Bugs Patrón
- Seguir señales patrón visual para movimiento brazo/pierna
- Mantener estabilidad core mientras procesando información
- Combina carga cognitiva con ejercicio físico
- Enfoque en precisión y control
- **Referencia Video**: [Jeff Nippard - Entrenamiento Abdominales](https://www.youtube.com/watch?v=_xdOuqokcm4)

### Planchas Reacción
- Mantener plancha respondiendo a señales externas
- Puede involucrar movimientos brazo/pierna según comando
- 30 segundos con desafíos cognitivos
- Mantener estabilidad plancha durante todo
- **Referencia Video**: [Jeff Nippard - Entrenamiento Core](https://www.youtube.com/watch?v=_xdOuqokcm4)

### Sprints Decisión
- Esperar señal luego sprintar en dirección indicada
- Esfuerzos máximos 10 segundos
- Combina tiempo reacción con salida física
- Simula toma decisiones partido bajo fatiga
- Recuperación completa para claridad mental

### Cambios Dirección Cognitivos
- Responder a señales visuales/auditivas para dirección
- 5 cambios dirección por serie
- Combina agilidad física con procesamiento mental
- Enfoque en respuestas rápidas precisas
- Mantener equilibrio y control

## Notas Rendimiento
- **Carga Cognitiva**: Desafíos mentales durante ejercicio físico
- **Entrenamiento Decisión**: Tiempo reacción mejorado bajo fatiga
- **Afilado Neural**: Coordinación cerebro-cuerpo mejorada
- **Aplicación Tenis**: Mejor toma decisiones durante partidos
