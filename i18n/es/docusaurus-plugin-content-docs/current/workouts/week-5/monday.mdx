# Lunes – Potencia Tren Inferior y Tenis Explosivo

## Horario Lunes

| Hora | Ejercicio/Ejercicio | Series×Reps | Instrucciones |
|------|---------------|-----------|--------------|
| 07:15–08:15 | Ejercicios Línea Base Tenis Explosivos | - | Trabajo pies agresivo, golpes fondo potentes |
| 08:25–09:35 | Sentadilla con Barra | 4×5 | Carga más pesada, descenso controlado, ascenso explosivo |
| | Sentadilla Búlgara | 3×8 por pierna | Enfoque en estabilidad y forma controlada |
| | Peso Muerto Rumano | 4×6 | Carga pesada, mantener columna neutral |
| | Elevaciones Pantorrilla Pesadas | 3×12 | Tempo lento, movimientos controlados |

| | Press Pallof Rotacional | 3×12 por lado | Rotación controlada, core estable |

## Instrucciones Rápidas Ejercicios

**Sentadilla con Barra (4×5)**: Retorno intensidad Semana 5, más pesada que Semana 3, intención explosiva

**Sentadilla Búlgara (3×8)**: Retorno a volumen completo, agregar carga desde Semana 3

**Peso Muerto Rumano (4×6)**: Aumentar a 4 series, énfasis carga pesada

**Sentadilla Española (3×60 seg)**: Progresión tiempo mantén pico, desafío máximo

**Press Pallof Rotacional (3×12)**: Aumentar reps, mantener control rotación

**Línea Base Tenis Explosiva**: Retorno potencia post-deload, patrones movimiento agresivos

## Puntos Enfoque Clave

- **Aumento Intensidad**: Retorno post-deload a entrenamiento alta intensidad
- **Desarrollo Potencia**: Ejercicios línea base explosivos con trabajo pies agresivo
- **Sentadillas Pesadas**: Carga aumentada (4×5) con énfasis ascenso explosivo
- **Aumento Volumen**: Peso muerto rumano aumentado a 4×6
- **Isométricos Extendidos**: Mantenes sentadilla española extendidos a 60 segundos
- **Potencia Core**: Press Pallof rotacional para fuerza core específica tenis

## Notas

Semana 5 comienza la fase Desarrollo con aumento intensidad después deload, enfocándose en desarrollo potencia y cargas más pesadas.
