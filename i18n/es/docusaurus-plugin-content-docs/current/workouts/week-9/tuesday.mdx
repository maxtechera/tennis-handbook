## Sesión Entrenamiento Complejo

### Calentamiento (15 minutos)
- Círculos brazo y trabajo banda
- Lanzamientos balón medicina (ligero)
- Activación escapular
- Variaciones flexión (5 cada tipo)

### Entrenamiento Principal - Pares Complejos Tren Superior (45 minutos)

**Par Complejo 1: Empuje Horizontal**
- Press Banca: 3 x 3 @ 85% 1RM
- Flexiones Pliométricas: 3 x 5 (explosivas)
- Descanso: 3 minutos entre pares

**Par Complejo 2: Jalón Vertical**
- Dominadas con Peso: 3 x 3 @ 85% 1RM
- Golpe Balón Medicina: 3 x 5 (velocidad máxima)
- Descanso: 3 minutos entre pares

**Par Complejo 3: Potencia Rotacional**
- Leñador Cable: 3 x 3 cada lado @ pesado
- Lanzamiento Balón Medicina Rotacional: 3 x 5 cada lado
- Descanso: 3 minutos entre pares

### Enfriamiento (15 minutos)
- Remo ligero o bicicleta brazo
- Estiramiento tren superior
- Trabajo estabilidad hombro

## Instrucciones Rápidas Ejercicios

### Flexiones Pliométricas
- Comenzar en posición flexión
- Bajar con control
- Explotar arriba para que manos salgan del suelo
- Aterrizar suavemente e inmediatamente repetir
- Enfoque en salida potencia máxima

### Golpe Balón Medicina
- Sostener balón arriba con brazos extendidos
- Activar core y golpear balón abajo con fuerza
- Atrapar en rebote o recoger
- Reiniciar postura antes próxima rep
- Velocidad máxima cada rep

### Lanzamiento Balón Medicina Rotacional
- Pararse de lado a pared, 3-4 pies lejos
- Sostener balón a nivel cadera
- Rotar explosivamente y liberar balón a pared
- Atrapar rebote e inmediatamente repetir
- Enfoque en secuencia cadera-a-hombro

### Leñador Cable
- Configurar cable a altura hombro
- Pararse perpendicular a cable
- Jalar cable a través cuerpo diagonalmente
- Controlar el retorno
- Mantener core activado durante todo

## Notas Rendimiento
- **Entrenamiento Complejo**: Ejercicio fuerza pesada seguido inmediatamente por movimiento explosivo
- **Períodos Descanso**: Recuperación completa entre pares es crucial
- **Intensidad**: Ambos ejercicios deben ser esfuerzo máximo
- **Volumen**: Reps menores para mantener calidad
