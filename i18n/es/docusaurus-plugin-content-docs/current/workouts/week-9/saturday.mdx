# Sábado - Semana 9

_Fase Entrenamiento Complejo - Competición Integrada_

## Opciones Sábado

### Opción A: Juego Partido Complejo (Muy Recomendado)
- **Calentamiento Complejo**: 20 minutos preparación multi-sistema
- **Partido Integrado**: Aplicar entrenamiento complejo en competición
- **Recuperación Compleja**: Protocolos recuperación multi-sistema

### Opción B: Acondicionamiento Complejo
- **Intervalos Multi-Sistema**: 25 minutos acondicionamiento complejo
- **Movilidad Integrada**: 20 minutos patrones movimiento complejos
- **Integración Sistema**: 10 minutos respiración y recuperación

## Instrucciones Rápidas Ejercicios

### Calentamiento Complejo
- 20 minutos preparación multi-sistema
- Combinar elementos aeróbicos, fuerza, y potencia
- Aumento complejidad progresivo
- Preparación mental para rendimiento integrado
- Activación sistema completo
- **Referencia Video**: [Jeff Nippard - Guía Calentamiento](https://www.youtube.com/watch?v=8wPpK6xW9s8)

### Intervalos Multi-Sistema (Día Sin Partido)
- 6×2 minutos intervalos complejos
- Combinar diferentes tipos movimiento dentro cada intervalo
- Incluir elementos aeróbicos, anaeróbicos, y potencia
- 2 minutos recuperación entre intervalos
- Aplicar principios entrenamiento complejo
- **Referencia Video**: [Jeff Nippard - Guías Cardio](https://www.youtube.com/watch?v=Kl0ht3ow8TQ)

### Movilidad Integrada (20 minutos)
- Combinar fuerza, flexibilidad, y movimiento
- Movimientos dinámicos con resistencia
- Patrones movimiento multi-planares
- Combinaciones movimiento complejas
- Integrar múltiples sistemas
- **Referencia Video**: [Jeff Nippard - Movilidad](https://www.youtube.com/watch?v=8wPpK6xW9s8)

### Integración Sistema (10 minutos)
- Ejercicios respiración que integran sistemas
- Recuperación que prepara múltiples sistemas
- Integración mental entrenamiento complejo
- Preparar para fase afilado neural
- Coordinación multi-sistema

### Recuperación Compleja (Día Partido)
- Protocolos recuperación multi-sistema
- Abordar recuperación cardiovascular, muscular, y neural
- Enfoque integrado al cuidado post-partido
- Estrategias recuperación complejas
- Preparar todos sistemas para próxima semana

## Notas Rendimiento
- **Enfoque Integración**: Múltiples sistemas trabajando juntos
- **Aplicación Compleja**: Aplicar entrenamiento en situaciones partido
- **Coordinación Sistema**: Todos elementos entrenamiento combinados
- **Preparación Fase**: Listo para afilado neural
