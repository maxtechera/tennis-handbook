## Instrucciones Rápidas Ejercicios

### Sprints Potencia Cancha (8×15 seg)
- Intensidad sprint máxima
- Largo cancha tenis si está disponible
- Enfoque en aceleración y velocidad máxima
- Recuperación completa entre sprints
- Simular demandas sprint partido

### Peso Muerto Barra Trap (5×3 @ 90%)
- Carga pesada para fuerza máxima
- Intención explosiva en cada rep
- Forma perfecta requerida
- Solo 3 reps para enfoque potencia
- **Referencia Video**: [Jeff Nippard - Guía Peso Muerto](https://www.youtube.com/watch?v=op9kVnSso6Q)

### Saltos Cajón (Altura Máxima)
- Cajón más alto que puedas saltar seguramente
- Extensión triple explosiva
- Aterrizaje suave en cajón
- Bajar caminando entre reps
- Altura máxima cada salto
- **Referencia Video**: [Jeff Nippard - Entrenamiento Pliométrico](https://www.youtube.com/watch?v=tcY1dI5-Teo)

### Empuje Cadera (5×5 @ 85%)
- Carga pesada con intención explosiva
- Velocidad impulso cadera máxima
- Activación glúteo completa
- Controlar fase excéntrica
- Enfoque desarrollo potencia
- **Referencia Video**: [Jeff Nippard - Guía Empuje Cadera](https://www.youtube.com/watch?v=xDmFkJxPzeM)

### Saltos Amplios (Distancia Máxima)
- Salto horizontal explosivo
- Usar brazos para impulso
- Aterrizar con rodillas suaves
- Medir distancia para progresión
- Esfuerzo máximo cada rep
- **Referencia Video**: [Jeff Nippard - Entrenamiento Pliométrico](https://www.youtube.com/watch?v=tcY1dI5-Teo)

### Push Press (4×4 @ 85%)
- Carga pesada con impulso explosivo
- Generar potencia desde piernas
- Velocidad barra rápida ascendente
- Posición recepción estable
- Velocidad máxima intencionada
- **Referencia Video**: [Jeff Nippard - Guía Push Press](https://www.youtube.com/watch?v=iaBVSJm78ko)

## Notas Rendimiento
- **Cargas Máximas**: 85-90% 1RM para componente fuerza
- **Expresión Potencia**: Movimiento explosivo inmediatamente después fuerza
- **Recuperación Completa**: Descanso completo entre series potencia
- **Enfoque Calidad**: Técnica perfecta en cargas altas
