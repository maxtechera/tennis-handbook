# Semana 8 - Deload y Renovación Táctica

## Objetivos Semana 8 (Segunda Semana Deload)

Descarga fatiga sistémica, alivio tendones, y afilado táctico antes próximo bloque intensificación. Esta semana se enfoca en:

- **Recuperación Sistémica**: ~50% reducción volumen gimnasio para descarga fatiga
- **Alivio Tendones**: Carga reducida para recuperación tejido conectivo
- **Afilado Táctico**: Intensidad tenis 60-70% con enfoque técnico
- **Mejora Movilidad**: Movilidad extra e integración trabajo respiración
- **Renovación Mental**: Estrés competitivo reducido para recuperación psicológica

## Horario Entrenamiento (Semana Deload)

- **Lunes**: Tren Inferior Ligero y Rally Fácil - 50% reducción carga + tenis cooperativo
- **Martes**: Salud Hombro y Mecánica Saque - Hombros a prueba balas + afinación saque
- **Miércoles**: Core y Descarga Aeróbica - Trabajo core ligero + aeróbico Zona-2
- **Jueves**: Movilidad + Golpeo Técnico Ligero - Flujo yoga + trabajo slice/toque
- **Viernes**: Preparación Neural y Juego Punto Corto - Pliométricos dosis baja + tie-breaks
- **Sábado**: Dobles Opcional o Caminar - Dobles divertidos ≤70% o caminar fácil
- **Domingo**: Descanso Completo - Dormir más, enfoque recuperación completa
