## Instrucciones Rápidas Ejercicios

### Peso Muerto Barra Trap (6×3 @ 92%)
- Carga muy pesada (92% 1RM)
- Enfoque fuerza máxima
- Forma perfecta requerida
- Intención explosiva a pesar carga pesada
- 3 reps para fuerza-potencia
- **Referencia Video**: [Jeff Nippard - Guía Peso Muerto](https://www.youtube.com/watch?v=op9kVnSso6Q)

### Saltos Profundidad (Reactivos)
- Altura cajón 18-24 pulgadas
- Bajar, aterrizar, saltar inmediatamente
- Tiempo contacto suelo mínimo
- Expresar fuerza como potencia reactiva
- Enfoque en ciclo estiramiento-acortamiento
- **Referencia Video**: [Jeff Nippard - Entrenamiento Pliométrico](https://www.youtube.com/watch?v=tcY1dI5-Teo)

### Empuje Cadera (5×4 @ 90%)
- Carga muy pesada para fuerza máxima
- Intención impulso cadera explosivo
- Activación glúteo completa
- Controlar excéntrico lentamente
- Puente fuerza a potencia
- **Referencia Video**: [Jeff Nippard - Guía Empuje Cadera](https://www.youtube.com/watch?v=xDmFkJxPzeM)

### Push Press (5×3 @ 90%)
- Carga fuerza máxima
- Generar potencia desde piernas
- Pesado pero intención explosiva
- Posición arriba perfecta
- Fuerza con componente velocidad
- **Referencia Video**: [Jeff Nippard - Guía Push Press](https://www.youtube.com/watch?v=iaBVSJm78ko)

### Flexiones Explosivas
- Expresión potencia peso corporal
- Manos salen del suelo en empuje
- Aterrizar suavemente y controlar
- Expresar potencia tren superior
- Prioridad forma perfecta
- **Referencia Video**: [Jeff Nippard - Variaciones Flexión](https://www.youtube.com/watch?v=Ae-sOQhp0wQ)

### Carga Granjero Pesada (60m)
- Carga más pesada posible para distancia
- Mantener postura perfecta
- Aplicación fuerza funcional
- Estabilidad core bajo carga
- Enfoque sobrecarga progresiva
- **Referencia Video**: [Jeff Nippard - Caminata Granjero](https://www.youtube.com/watch?v=p8FWv6bUJ7w)

## Notas Rendimiento
- **Cargas Máximas**: 90-92% 1RM para fuerza pura
- **Puente Fuerza-Potencia**: Levantamiento pesado seguido por movimiento explosivo
- **Descanso Extendido**: 4-5 minutos para recuperación completa
- **Forma Perfecta**: Crítica en cargas máximas
