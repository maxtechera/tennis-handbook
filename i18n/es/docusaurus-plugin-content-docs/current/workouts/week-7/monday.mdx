# Lunes – Potencia Tren Inferior y Explosividad Línea Base

## Horario Lunes

| Hora | Ejercicio/Ejercicio | Series×Reps | Instrucciones |
|------|---------------|-----------|--------------|
| 07:15‑08:15 | Tenis – Línea Base Reactiva | 4×6‑peloteo ráfaga rally | Explotar primer paso, split‑step activo |
| 08:25‑09:30 | Sentadilla trasera (cluster) | 4×(3+2) @ 85% | 3 reps, 15s descanso, luego 2 reps. Enfoque velocidad barra |
| | Salto zancada pie trasero elevado | 3×5/pierna | Peso corporal, contacto suelo mínimo |
| | Peso muerto piernas rígidas | 3×6 | Pesado pero nítido; 2s excéntrico |
| | ISO pantorrilla de pie | 3×30s | Mantén medio pie, respirar lentamente |
| 09:30‑09:50 | Sentadilla española iso + Leñador cable | 2×45s / 2×12/lado | Mantener rigidez tronco |

## Instrucciones Rápidas Ejercicios (Fase Intensificación)

**Cluster Sentadilla Trasera (4×(3+2))**: 3 reps al 85%, descanso 15s, luego 2 reps más - enfoque en velocidad barra

**Salto Zancada Pie Trasero Elevado**: Salto explosivo entre piernas, tiempo contacto suelo mínimo

**Peso Muerto Piernas Rígidas (Pesado)**: Excéntrico 2 segundos, carga pesada, mantener posición pierna durante todo

**ISO Pantorrilla de Pie**: Mantener posición rango medio por 30s, enfoque en respiración controlada

**Línea Base Reactiva Tenis**: Rallies ráfaga 6 pelotas con primeros pasos explosivos

**Enfoque Cluster**: Desarrollo potencia a través metodología entrenamiento descanso-pausa

## Puntos Enfoque Clave

- **Tenis Reactivo**: Línea base explosiva con rallies ráfaga y split-steps activos
- **Entrenamiento Cluster**: Clusters sentadilla trasera para desarrollo potencia
- **Entrenamiento Salto**: Saltos zancada pie trasero elevado con contacto suelo mínimo
- **Peso Muerto Pesado**: Ejecución nítida con excéntrico controlado
- **Potencia Core**: Leñadores cable para fuerza rotacional

## Notas

Semana 7 enfatiza conversión potencia mientras mantiene fuerza pico con volumen reducido.
