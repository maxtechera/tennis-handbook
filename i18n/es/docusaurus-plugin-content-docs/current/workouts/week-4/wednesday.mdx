## Instrucciones Rápidas Ejercicios

### Peloteo Suave Tenis
- Mantener ritmo peloteo fácil y cómodo
- Enfocarse en mecánica golpe suave
- Esfuerzo físico mínimo
- Enfatizar consistencia sobre potencia
- Usar esto como terapia movimiento

### Rutina Movilidad y Estiramiento
- Mantener cada estiramiento 30-60 segundos
- Respirar profundamente durante mantenes
- Enfocarse en grupos musculares principales
- No forzar ninguna posición
- Incluir movilidad hombro, cadera, y tobillo

### Estabilidad Core (2×10 / 2×30 seg)
- Dead bugs o bird dogs
- Mantener posiciones con forma perfecta
- Respirar normalmente durante todo el ejercicio
- Enfocarse en control, no intensidad
- Parar si ocurre cualquier molestia

## Puntos Enfoque Clave (Semana Deload)

- **Tenis Suave**: Peloteo fácil con esfuerzo físico mínimo
- **Movilidad Extendida**: Estiramiento cuerpo completo y preparación movimiento
- **Trabajo Core Ligero**: Ejercicios estabilidad core intensidad mínima
- **Enfoque Recuperación**: Énfasis en calidad movimiento y circulación
- **Reducción Estrés**: Entrenamiento baja demanda para recuperación sistema nervioso

## Notas

Miércoles sirve como día recuperación activa con movimiento suave y trabajo movilidad extensivo para apoyar proceso deload.
