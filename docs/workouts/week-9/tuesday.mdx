## Complex Training Session

### Warm-up (15 minutes)
- Arm circles and band work
- Medicine ball throws (light)
- Scapular activation

### Main Training - Upper Body Complex Pairs (45 minutes)

**Complex Pair 1: Horizontal Push**
- Bench Press: 3 x 3 @ 85% 1RM
- Plyometric Push-ups: 3 x 5 (explosive)
- Rest: 3 minutes between pairs

**Complex Pair 2: Vertical Pull**
- Weighted Pull-ups: 3 x 3 @ 85% 1RM
- Medicine Ball Slam: 3 x 5 (maximum velocity)
- Rest: 3 minutes between pairs

**Complex Pair 3: Rotational Power**
- Cable Woodchop: 3 x 3 each side @ heavy
- Rotational Medicine Ball Throw: 3 x 5 each side
- Rest: 3 minutes between pairs

### Cool-down (15 minutes)
- Light rowing or arm bike
- Upper body stretching
- Shoulder stability work

## Quick Exercise Instructions

### Plyometric Push-ups
- Start in push-up position
- Lower with control
- Explode up so hands leave ground
- Land softly and immediately repeat
- Focus on maximum power output

### Medicine Ball Slam
- Hold ball overhead with extended arms
- Engage core and slam ball down forcefully
- Catch on bounce or pick up
- Reset posture before next rep
- Maximum velocity each rep

### Rotational Medicine Ball Throw
- Stand sideways to wall, 3-4 feet away
- Hold ball at hip level
- Rotate explosively and release ball at wall
- Catch rebound and immediately repeat
- Focus on hip-to-shoulder sequence

### Cable Woodchop
- Set cable at shoulder height
- Stand perpendicular to cable
- Pull cable across body diagonally
- Control the return
- Keep core engaged throughout

## Performance Notes
- **Complex Training**: Heavy strength exercise immediately followed by explosive movement
- **Rest Periods**: Full recovery between pairs is crucial
- **Intensity**: Both exercises should be maximum effort
- **Volume**: Lower reps to maintain quality
